# Task Management System

## Project Description

Task Management System is a web application that helps users create, manage, update, and track their daily tasks efficiently.

## Technologies Used

* Java
* Spring Boot
* JSP
* MySQL
* HTML
* CSS

## Features

* Add new tasks
* Update tasks
* Delete tasks
* View task list
* User-friendly interface

## Project Structure

* src – Java source code
* pom.xml – Maven dependencies
* README.md – Project documentation

## Author

Developed as a Full Stack Java project.
